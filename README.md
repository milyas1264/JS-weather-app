# 🌦️ Weather App

A simple and responsive Weather Application built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the OpenWeather API.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Display temperature, humidity, and wind speed
* 📍 Weather icons based on conditions
* 🕘 Recent searches (stored in localStorage)
* ❌ Delete individual recent searches
* ⚡ Fast and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeather API

---

## 🔑 Setup Instructions

1. Get your API key from OpenWeather
2. Open `app.js`
3. Replace the API key:

```javascript
const apiKey = "YOUR_API_KEY";
```
